# Simple Task Tracker (FastAPI) 🚀

Асинхронное REST API для управления списком задач. Проект создан с использованием современного стека технологий Python и демонстрирует навыки построения масштабируемой архитектуры.

## 🛠 Стек технологий
* **Python 3.14+** (экспериментальные возможности)
* **FastAPI** — веб-фреймворк для создания API.
* **SQLAlchemy 2.0 (Async)** — ORM для работы с базой данных.
* **Alembic** — система миграций для БД.
* **Pydantic v2** — валидация данных и схемы ответов.
* **SQLite (aiosqlite)** — легкая асинхронная база данных.
* **python-dotenv** — управление конфигурацией.

## 🌟 Ключевые особенности
- **Полный CRUD**: создание, получение (списком и по ID), частичное обновление (PATCH) и удаление задач.
- **Миграции**: управление структурой БД через Alembic.
- **Асинхронность**: неблокирующие запросы к БД для высокой производительности.
- **Документация**: Автоматическая интерактивная документация Swagger/OpenAPI.

## 🚀 Как запустить проект локально

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/ваш-логин/SimpleTaskTracker.git](https://github.com/ваш-логин/SimpleTaskTracker.git)
   cd SimpleTaskTracker
    ```
   
2. **Установите зависимости**:
    ```bash 
    pip install -r requirements.txt 
    ```
     
3. **Запустите приложение**:
    ```bash
     uvicorn app.main:app --reload  
     ```
